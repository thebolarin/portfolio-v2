Selected Projects

Currency Converter

currencyconverter.com

The project involves creating a currency converter web page using HTML, CSS, and ECMAScript. The page has a user-friendly interface that prompts the user to input the source and destination currencies, displays the current exchange rate, calculation timestamp, and the date of the last exchange rate update.

The user is able to enter a valid amount for the source currency, and the page will accurately calculate the amount for the destination currency.

Fuel Forecaster

fuelforcaster.com

This project involves designing, implementing, and testing a forecasting page for heating fuel costs in the UK, using actual data from the UK government's Department for Business, Energy & Industrial Strategy (BEIS). The data is provided in an Excel spreadsheet, and the project requires creating a page using HTML, CSS, and ECMAScript.

The page has an interface displaying a chart representing a twelve-month window of the prices for four basic fuel types (solid fuels, gas, electricity, and liquid fuels). The user can scroll through the data, filter the chart view for different fuel types, and use the straight-line forecasting method to predict costs for the next 12 months. Charting libraries like Chart.js was used.
